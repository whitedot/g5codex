22.

~~~text
# AGENTS.md instructions for C:\laragon\www\g5

<INSTRUCTIONS>
# AGENTS.md

## 커밋 메시지 작성 방법

- 커밋 메시지는 한국어로 작성한다.
- 형식은 `접두사: 커밋 메시지`를 사용한다.
- 접두사는 변경 성격을 짧게 나타낸다.

## 보안 토큰 및 링크 키 작업 규칙

- CSRF 토큰, 인증 메일 토큰, 비밀번호 재설정 nonce, 자동로그인 키처럼 요청 검증에 쓰이는 값은 `rand()`, `mt_rand()`, `uniqid()`, 단순 `md5()`로 만들지 않는다.
- 새 난수 토큰은 `random_bytes()` 또는 `random_int()` 기반 helper를 사용한다.
- 토큰, nonce, 링크 키, 자동로그인 키 비교는 `hash_equals()` 또는 프로젝트의 상수시간 비교 helper를 사용한다.
- 비밀값이 필요한 검증 키는 단순 해시 대신 HMAC 기반 helper를 사용한다.

예:

```text
정리: 런타임에 불필요한 파일 제거
설정: 운영 환경 설정 파일 분리
수정: 회원가입 간편인증 사용 차단
문서: 배포용 README 추가
```

</INSTRUCTIONS>
~~~
